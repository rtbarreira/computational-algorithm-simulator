# Computational Algorithm Simulator
This is a repository containing the implementation of my master`s project from Universidade de São Paulo (USP).

Created by Raíssa Barreira.

## Installation

1. Install [Docker](https://docs.docker.com/engine/install/)

2. Run `docker compose up -d --build`

## Usage

- The API exposed through Docker is available on http://localhost:5000/
- The N8N instance is available on http://localhost:5678/
